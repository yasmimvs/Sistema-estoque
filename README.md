
 Sistema de Estoque - ASP.NET + Firebase

sistema simples de gerenciamento de estoque, desenvolvido em ASP.NET WebForms com integração ao Firebase Firestore.


 Funcionalidades:

- Adicionar produtos
- Listar produtos em tabela
- Excluir produtos
- Integração com banco de dados em nuvem (Firebase)
  

 Ferramentas utilizadas:

- ASP.NET WebForms (C#)
- HTML / CSS
- Firebase Firestore
- HttpClient (API REST)

Visual do sistema: ( print nos arquivos)

 Objetivo:

Projeto criado para estudo e portfólio, simulando um sistema real de controle de estoque.
